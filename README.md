# Magicq-Akai-LPO8
Control MagicQ faders in the demo version using Akai LPO8 midi controller utilizing AutoHotkey

Built using Claude, so do not expect for this to be stable. This has worked and not crashed for me, but your mileage may vary.
This is a prototype as of 7.3.2026

#requierments Autohotkey V2
